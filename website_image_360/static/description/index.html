Website Product Image 360 view

Purpose of this module is to provide a 360 dimension view of a product on website.

User has to check Display 360 image boolean as true for adding all 360 dimensions images.

From the tab, all different dimensioned images of a product are added which isto be viewed on website.

By clicking on the 360 view, the image gets rotate at 360 dimensions on mouse movement.

Video has been added to view the product image rotating in 360 dimension view.
